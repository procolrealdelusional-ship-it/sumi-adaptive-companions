# SUMI DOJO - Roblox Scripts

## Setup (2 steps only)

### Step 1: Server Script
1. Open Roblox Studio
2. Go to ServerScriptService
3. Add new Script
4. Paste contents of `SUMI_MasterLoader.lua`

### Step 2: Client Script
1. Go to StarterPlayer > StarterPlayerScripts
2. Add new LocalScript
3. Paste contents of `SUMI_ClientUI.lua`

## What gets built automatically:
- Dojo floor (wood planks)
- Torii gate with SUMI DOJO sign
- 6 World portals (Zen Garden, Fire Dojo, Ocean Temple, Thunder Peak, Bamboo Forest, Volcano Core)
- 4 NPC companions (Dee Dee, Koko, Mochi, Blaze)
- 3x3 drill tile grid
- DataStore persistence (XP, streak, YinYang)
- Remote events for client-server communication
- HUD with XP, streak, balance display
- Companion dialog system
- Notification popups

## Features:
- 3-drill sessions (hook design)
- Tri-language: English / Punjabi / Punglish
- YinYang energy balance
- Streak tracking with daily reset
- Fibonacci XP scaling
- Leaderboard system
- Parent dashboard data
